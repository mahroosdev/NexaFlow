Product support

Resolve setup and connection issues.

Start with the checks below. Submit a public support ticket only when the problem remains.

Fast checks

Most problems can be isolated in a few steps.

Windows setup

Confirm Windows 10 or 11, run the current v1.0.0 installer, and accept the first firewall request when Remote Access starts.

Android pairing

Put the phone and PC on the same Wi-Fi or phone hotspot. Enter the address shown by NexaFlow Desktop and approve the named device.

Remote screen

Keep Remote Access on, verify the paired status, then try Balanced quality before Sharp on slower local networks.

Chrome extension

Extract the ZIP before choosing Load unpacked. Select the folder that directly contains manifest.json.

File verification

Use the checksum page to confirm the setup package was downloaded without corruption or substitution.

Open a support issue

Describe the problem.

Support tickets are public. Remove passwords, pairing tokens, private screenshots, IP addresses, and workflow data before submitting.
