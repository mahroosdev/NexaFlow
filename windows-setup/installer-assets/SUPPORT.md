# NexaFlow Support

Use the NexaFlow support page for help, bug reports, install questions, and pairing issues:

```text
https://web-nexaflow.netlify.app/support
```

Support requests are handled through the public GitHub Issues page:

```text
https://github.com/mahroosdev/NexaFlow/issues
```

GitHub issues are public. Do not include passwords, pairing tokens, screenshots, or sensitive workflow data.

## Before Requesting Help

Please include:

- NexaFlow version from `VERSION`
- Windows version
- Whether you installed with `NexaFlow_Setup.exe` or ran `NexaFlow.exe` directly
- A short description of what you expected and what happened
- Any error text shown by NexaFlow or the installer
