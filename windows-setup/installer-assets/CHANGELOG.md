# NexaFlow Changelog

## 1.0.0

- Prepared NexaFlow for the first public website/private release.
- Added consent-based Remote Access with desktop Start/Stop controls, approval requests, local address, and connected-device status.
- Added Android companion support for local pairing, saved-workflow playback, stable screen monitoring, and landscape zoom.
- Added Chrome extension companion for browser workflow recording, replay, export/import, and desktop quick controls.
- Improved Android startup polish, safe-area spacing, responsive layout, Settings, and light/dark theme support.
- Improved Windows Settings, timezone apply behavior, About version placement, taskbar/icon handling, and release packaging.
- Restricted companion control to the local network or a personal phone hotspot.
- Updated release checksums, privacy/support documents, launch notes, and website download documentation.
